Windows 11 is Microsoft's operating system that runs your computer. This guide covers the basics and how to fix common problems.

---

## 1. Getting Started

* **Check Compatibility:** Your PC needs specific hardware. Use [PC Health Check app](https://aka.ms/GetPCHealthCheckApp) to verify.
* **Upgrade:** If you have Windows 10, go to Settings → Windows Update → Check for updates. Free upgrade if compatible.
* **First Setup:** Create or sign in with Microsoft account. Choose privacy settings. Set up PIN for quick login.

---

## 2. Navigating Windows 11

## The Start Menu

Click the Windows icon (centered at bottom) or press Windows key on keyboard.

* **Pinned apps:** Your most-used programs
* **Recommended:** Recent files and apps
* **All apps:** Complete list of installed programs
* **Power button:** Shut down, restart, or sleep

## The Taskbar

Bottom bar shows open programs and shortcuts.

* **Search icon:** Find files, apps, or settings (or just start typing)
* **Widgets:** News, weather, calendar at a glance
* **Task View:** See all open windows and virtual desktops
* **Quick Settings:** Volume, Wi-Fi, Bluetooth (bottom right)
* **Notification Center:** Messages and alerts (click date/time)

## File Explorer

Your file manager—where documents, pictures, and downloads live.

* **Quick Access:** Recent files and frequently used folders
* **This PC:** All drives and main folders (Documents, Pictures, Downloads, etc.)
* **OneDrive:** Cloud storage (if signed in with Microsoft account)
* **Navigation:** Click folders to open, use back/forward arrows

## 3. Essential Settings

Press Windows key + I to open Settings, or click Start → Settings.

## Personalization

Make Windows look the way you want.

* **Background:** Change desktop wallpaper
* **Colors:** Pick accent color and light/dark mode
* **Themes:** Coordinated color schemes
* **Start:** Customize what shows in Start menu
* **Taskbar:** Choose which icons appear

## System

Control core features and displays.

* **Display:** Screen brightness, resolution, multiple monitors
* **Sound:** Volume, input/output devices
* **Notifications:** Control which apps can send alerts
* **Power:** Battery settings, sleep timers
* **Storage:** See what's using space, cleanup tools

## Accounts

Manage your profile and sign-in.

* **Your info:** Profile picture, account details
* **Sign-in options:** Password, PIN, fingerprint, facial recognition
* **Family:** Set up accounts for kids with parental controls

## Windows Update

Keep your PC secure and running smoothly.

* Check for updates regularly (happens automatically by default)
* Install updates when prompted
* Restart when needed—don't skip this
* Updates include security fixes and new features

## 4. Working with Apps

## Installing Apps

**Microsoft Store (recommended):**

* Click Start → Microsoft Store
* Search for app you want
* Click Get or Install
* Safe, verified apps

**From the web:**

* Download installer file (.exe)
* Double-click to run
* Follow installation steps
* **Be careful:** Only download from trusted sources

## Uninstalling Apps

* Settings → Apps → Installed apps
* Find app in list
* Click three dots → Uninstall
* Or right-click app in Start menu → Uninstall

## Default Apps

Choose which apps open which file types.

* Settings → Apps → Default apps
* Select app or file type
* Choose your preferred program
* Example: Set Chrome or Edge as default browser

## 5. Managing Files

## Creating Folders

Keep files organized.

* Open File Explorer
* Right-click empty space → New → Folder
* Name your folder
* Drag files into it

## Moving Files

* **Cut and paste:** Right-click file → Cut → Navigate to destination → Right-click → Paste
* **Drag and drop:** Click file, hold, drag to folder
* **Copy instead of move:** Hold Ctrl while dragging, or use Copy instead of Cut

## Finding Lost Files

* Press Windows key and start typing filename
* Or use Search in File Explorer (top right)
* Check Downloads folder (common location)
* Check Recycle Bin if accidentally deleted

## Backing Up Files

Don't lose important stuff.

* **OneDrive:** Automatic cloud backup (if enabled)
* **External drive:** Copy important folders to USB drive or external hard drive
* **File History:** Settings → System → Storage → Advanced storage settings → Backup options

## 6. Troubleshooting

Having problems? Try these steps first.

## Computer Running Slow

**Quick fixes:**

1. **Restart:** Solves many issues. Start → Power → Restart
2. **Close unused programs:** Right-click taskbar → Task Manager → End programs using lots of resources
3. **Check storage:** Low disk space slows things down
4. **Run maintenance:** Settings → System → Storage → Cleanup recommendations

**Deeper solutions:**

* Disable startup programs: Settings → Apps → Startup
* Check for malware: Windows Security → Virus & threat protection → Quick scan
* Update drivers: Settings → Windows Update → Advanced options → Optional updates

## Internet Not Working

1. Check Wi-Fi is on: Quick Settings → Wi-Fi
2. Forget and reconnect to network
3. Restart router (unplug 30 seconds)
4. Run troubleshooter: Settings → Network & internet → Advanced network settings → Network troubleshooter
5. Check other devices—is it just your PC or whole network?

## App Won't Open or Crashes

1. **Restart app:** Close completely (Task Manager if needed) and reopen
2. **Update app:** Microsoft Store → Library → Get updates
3. **Repair app:** Settings → Apps → Installed apps → App → Advanced options → Repair
4. **Reset app:** Same location as Repair, but use Reset (loses app data)
5. **Reinstall:** Uninstall and download fresh copy

## Windows Update Issues

If updates fail or cause problems:

* Try updating again—sometimes works second time
* Check internet connection
* Free up disk space
* Run Windows Update troubleshooter: Settings → System → Troubleshoot → Other troubleshooters → Windows Update
* [Check for known issues](https://learn.microsoft.com/en-us/windows/release-health/)

## Blue Screen (BSOD)

If computer crashes with blue screen:

* Note error code if shown
* Restart computer
* If happens repeatedly, search error code for solutions
* Check for Windows updates
* May need professional help if persistent

## 7. Tips & Tricks

## Keyboard Shortcuts

Speed up common tasks.

* **Windows key:** Open Start menu
* **Windows + D:** Show desktop (minimize everything)
* **Windows + E:** Open File Explorer
* **Windows + I:** Open Settings
* **Windows + L:** Lock computer
* **Windows + Tab:** Open Task View
* **Alt + Tab:** Switch between open programs
* **Windows + Shift + S:** Take screenshot (Snipping Tool)
* **Ctrl + C:** Copy
* **Ctrl + V:** Paste
* **Ctrl + Z:** Undo
* **Ctrl + F:** Find/search

## Snap Layouts

Organize windows side by side.

* Hover over maximize button (top right of window)
* Choose layout from popup
* Select which window goes in each space
* Great for multitasking

## Virtual Desktops

Create separate workspaces.

* Click Task View (taskbar) or press Windows + Tab
* Click New desktop
* Switch between desktops for different projects
* Example: One for work, one for personal

## Widgets

Quick info without opening apps.

* Click Widgets icon on taskbar
* See weather, news, calendar, stocks
* Customize what appears: Widgets → Settings (top right)
* Pin important widgets to top

## Focus Assist

Block distracting notifications.

* Quick Settings → Focus assist
* Priority only: Only important notifications
* Alarms only: Complete silence except alarms
* Schedule quiet times automatically

## Dark Mode

Easier on eyes in low light.

* Settings → Personalization → Colors
* Choose Dark under "Choose your mode"
* Or Custom to mix light/dark for apps and system

## Night Light

Reduce blue light in evening.

* Settings → System → Display → Night light
* Schedule automatically (sunset to sunrise)
* Adjust intensity with slider

## Common Questions

## Do I need Windows 11?

Not necessarily. Windows 10 gets security updates until October 2025. Windows 11 has modern interface and new features, but Windows 10 still works great. Upgrade when ready, no rush.

## How much does Windows 11 cost?

Free upgrade from Windows 10 if your PC is compatible. New licenses cost $139 (Home) or $199 (Pro). Usually comes pre-installed on new computers.

## What's the difference between Home and Pro?

Most people need Home. Pro adds features for businesses: remote desktop, BitLocker encryption, domain joining. Unless you know you need Pro features, Home is fine.

## Can I go back to Windows 10?

Yes, within 10 days of upgrading: Settings → System → Recovery → Go back. After 10 days, would need clean install (more complicated). Keep Windows 10 backup just in case.

## Why does my PC keep updating?

Security and bug fixes. Updates protect against viruses and hacking. Can schedule updates: Settings → Windows Update → Advanced options → Active hours (PC won't restart during these times).

## How do I take screenshots?

Multiple ways: Press Windows + Shift + S for Snipping Tool (select area), Print Screen key copies whole screen, or Windows + Print Screen saves to Pictures → Screenshots folder.

## Can I customize the taskbar position?

No, Windows 11 taskbar stays at bottom (unlike Windows 10). Can align icons left or center: Settings → Personalization → Taskbar → Taskbar behaviors → Taskbar alignment.

## What's a Microsoft account and do I need one?

Email-based account for Microsoft services. Strongly recommended—enables OneDrive, syncs settings across devices, required for Microsoft Store. Can use local account instead but lose features.

## How do I uninstall Windows 11?

Within 10 days: Settings → System → Recovery → Go back. After 10 days: Need to reinstall Windows 10 from scratch using installation media. Backup files first.

## What is OneDrive?

Microsoft's cloud storage (like Google Drive or Dropbox). Free 5GB included with Microsoft account. Files saved to OneDrive folder sync across devices and accessible anywhere.

## Why is my storage full?

* Temporary files pile up over time
* Large apps and games
* Downloads folder never cleaned
* OneDrive may be syncing everything
* Run Storage Sense: Settings → System → Storage → Cleanup recommendations

## Can I remove pre-installed apps?

Some yes, some no. Right-click app → Uninstall works for most. Built-in Windows apps can't be removed but can be hidden from Start menu.

## Need Help?

**Can't sign in:**

* Check username and password
* Reset password at account.microsoft.com
* Try PIN instead of password
* Check Caps Lock isn't on

**Screen looks weird:**

* Check display resolution: Settings → System → Display
* Update graphics driver
* Try different scale setting
* Restart computer

**No sound:**

* Check volume isn't muted (speaker icon)
* Test different app
* Check correct output device selected
* Run audio troubleshooter

**Can't connect to Wi-Fi:**

* Check Wi-Fi is turned on
* Restart PC and router
* Forget network and reconnect
* Run network troubleshooter

**Need more help:**

* Windows Help: Press F1 in most apps
* Get Help app: Search "Get Help" in Start
* Microsoft Support: support.microsoft.com
* Community forums: answers.microsoft.com
* Reddit: r/Windows11 (community help)

**Remember:** Windows 11 might seem complicated at first, but you don't need to know everything right away. Focus on the basics—opening apps, managing files, and browsing the web. The rest you'll pick up as you go. Everyone was a beginner once! Don't be afraid to explore and click around. Most things can be undone, and you'll learn faster by trying things yourself. Take your time, be patient with yourself, and before you know it, you'll be navigating like a pro. Happy computing!
